<h2>Showdown Demo</h2>

<p>You can try out Showdown on this page:</p>

<ul>
<li>Type some <a href="http://daringfireball.net/projects/markdown/">Markdown</a> text on the left side.</li>
<li>See the corresponding HTML on the right.</li>
</ul>

<p>For a Markdown cheat-sheet, switch the right-hand window from <em>Preview</em> to <em>Syntax Guide</em>.</p>

<p>Showdown is a JavaScript port of the original Perl version of Markdown.  You can get the full <a href="http://attacklab.net/showdown/showdown-v0.9.zip">source code</a> by clicking on the version number at the bottom of the page.</p>

<p>Also check out <a href="http://wmd-editor.com/">WMD, the Wysiwym Markdown Editor</a>.  It'll be open source soon; email me at the address below if you'd like to help me test the standalone version.</p>

<p><strong>Start with a blank page or edit this document in the left window.</strong></p>
